## Notes

- The app fetches web content metadata (titles, thumbnails) from URLs the user provides.
- This metadata may theoretically include any web content, but the app itself has no content controls.
- If prompted about "user-generated content" or "website browsing", select **No** — the app saves links but does not render full web pages.
